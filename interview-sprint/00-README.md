# DE Interview Sprint — 30 Days

Parallel track to **CHALLENGES.md**. Focus: technical interviews (product companies, analytics engineer roles).

## How to Combine with the Lab

| Week | Lab (quest) | Sprint |
|------|-------------|--------|
| 1–2 | Level 1 SQL | 01-SQL days 1–10 |
| 3–4 | Level 2 Python | 01-SQL 11–20 + 02-Python |
| 5–6 | Level 3 dbt | 03-Modeling |
| 7+ | Level 4–5 | 04-Senior AE (overview) |

**10–15 hrs/week** ≈ 30-day sprint + 6–8 weeks for the core quest.

- Explain one solution out loud (2 min recording) — interview practice
